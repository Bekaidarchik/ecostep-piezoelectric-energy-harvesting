# EcoStep One-Page Outreach Summary

Project lead: Amangeldi Bekaidar  
Location: Almaty, Kazakhstan  
Status: working prototype, validation stage

## What EcoStep Is

EcoStep is a low-cost piezoelectric floor tile prototype that converts footsteps into measurable electrical signals. The current prototype uses a 4 x 4 array of 16 PZT piezoelectric elements inside a layered compression tile.

## Why It Matters

High-footfall spaces such as schools, malls, public walkways, and metro stations contain constant human movement. EcoStep investigates whether part of this mechanical energy can be captured, measured, and used for small sensors, indicators, or climate-awareness displays.

## Current Evidence

- Physical prototype built and photographed.
- 16 PZT elements arranged as a 4 x 4 array.
- Approximate tile size: 50 cm x 50 cm.
- Rough prototype cost assumption: about USD 75.
- Arduino logging and Python analysis workflow prepared.
- GitHub documentation package prepared.

## What Still Needs Validation

- Exact energy per step under known load.
- Capacitor charging performance.
- Single PZT vs full-array comparison.
- Exact wiring topology.
- Whether any demo electronics can be powered only by the tile.

## Pilot Ask

EcoStep is seeking feedback, measurement support, and controlled demonstration opportunities in Almaty. The first ask is not public deployment. The first ask is a careful conversation about safety, testing, permissions, and whether a small educational pilot could be possible later.
