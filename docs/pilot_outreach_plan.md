# EcoStep Pilot Outreach Plan

Status: prepared outreach plan, not a confirmed partnership

## Goal

Use EcoStep's prototype evidence to start careful conversations with Almaty schools, labs, public-space operators, and Almaty Metro contacts. The first goal is feedback and controlled measurement or demo permission, not immediate public deployment.

## Safe Outreach Ask

Ask for one of the following:

- 15-minute feedback meeting
- technical review of the measurement plan
- help with controlled voltage-time testing
- permission to demonstrate the prototype in a supervised setting
- referral to the correct facilities, engineering, or education contact

Do not ask for or announce:

- confirmed installation
- partnership
- public deployment
- verified energy output

## Target Categories

| Category | Reason | First ask |
| --- | --- | --- |
| Almaty Metro / public transport | High-footfall pilot relevance | Ask for feedback and the correct pathway for a future controlled demo |
| Schools | Educational climate-awareness fit | Ask for a supervised demonstration or STEM feedback |
| University / engineering labs | Measurement credibility | Ask for controlled testing support |
| Maker labs / electronics mentors | Hardware iteration | Ask for wiring, load-test, and oscilloscope feedback |
| Malls / public walkways | Future high-footfall setting | Ask for facilities feedback only |
| Climate / youth organizations | Communication and mentorship | Ask for public-impact feedback |

## Message Boundary

Use:

> EcoStep is a working prototype in validation stage. I am looking for feedback, measurement support, or the correct pathway for a controlled demonstration.

Avoid:

> EcoStep is ready to install in Almaty Metro.

## Follow-Up Rhythm

- Day 0: send short email with one-page summary.
- Day 5-7: send one polite follow-up.
- Day 14: if no response, mark as waiting or closed and move to another contact.
- If there is interest, ask what documentation, safety review, or permission process is required.
